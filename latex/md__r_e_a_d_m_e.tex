Gerencia\+Lab is a queue management system designed to optimize patient flow in clinical laboratories. It replaces manual paper-\/based ticketing with an automated solution that categorizes exams, prioritizes special needs patients (elderly, pregnant women, people with disabilities), and provides real-\/time queue tracking.\hypertarget{md__r_e_a_d_m_e_autotoc_md2}{}\doxysection{\texorpdfstring{Problem Statement}{Problem Statement}}\label{md__r_e_a_d_m_e_autotoc_md2}
Current laboratory operations rely on\+:
\begin{DoxyItemize}
\item Physical paper tickets with no exam type differentiation
\item First-\/come-\/first-\/served manual queues
\item No standardized priority system
\item Chaotic peak hour situations
\item No performance metrics collection
\end{DoxyItemize}

This leads to\+:
\begin{DoxyItemize}
\item Patient dissatisfaction
\item Inefficient resource allocation
\item Unmanaged wait times
\item Operational confusion
